# django-ecommerce
Aplicación de comercio online utilizando Django y SQLite

###Cómo ejecutar la app
1. Instala [python](https://www.python.org)
2. Clona el repositorio
3. Desde la linea de comandos ve hasta la ubicacion del repositorio 

    `cd django-ecommerce`
    
4. Crea un entorno virtual en la raiz del repositorio
     
     `virtualenv venv`
     
5. Activa el entorno virtual  

     `source venv/scripts/activate`

6. Instala los requerimientos en el entorno virtual

    `pip install -r requeriments.txt`

7. Desde la raiz del repositorio ejecuta el siguiente comando

    `python manage.py runserver`

8. En tu navegador ve a la direccion "127.0.0.1:8000"
